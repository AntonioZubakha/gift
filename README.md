# gift — Cat Chaos Quest

Статический сайт-квест **Cat Chaos: Theatre of the Absurd** (HTML, CSS, JS — без сборки). Абсурдистская игра про котов, еду, мусорку, собаку и немного ворожбы.

## Что внутри (безумного)

- **Combo-система**: кормишь котов быстро — получаешь `×3 RAMPAGE!`, `×5 KITTY GOD!`, `×10 ABSURD INFINITY!`
- **Мысли котов** в пузырях: философия, экзистенция, запрос на тунец
- **Случайные катаклизмы** каждые ~10 секунд: радужный режим, наклон мира, slow-motion, дождь эмодзи, смена палитр, пьяные коты
- **Party Mode 🪩** — нажми жёлто-розовую кнопку и встречай дискотеку
- **Chaos Dice 🎲** — вручную вызвать случайное безумие
- **Konami-код** `↑↑↓↓←→←→BA` → **MEGA PARTY** с бесконечными собаками
- **Клик 5 раз по заголовку** → **MEOW STORM** (8 котов с криком «МЯУ!»)
- **Сверкающий след** за едой при перетаскивании
- **CRT-сканлайны** + шум + виньетка
- **Chaos Meter** в HUD (😐 → ☠️ по мере увеличения абсурда)
- **Пасхалки в пароле**: введи `кот`, `cat`, `666`, `лол` — будут отдельные сообщения
- **Дискотечный шар** на экране победы

## GitHub Pages

1. Запушьте репозиторий в ветку **`main`** (корень — `index.html`, `style.css`, `script.js`, папка **`assets/`**).
2. В репозитории: **Settings → Pages → Build and deployment**.
3. В поле **Source** выберите **GitHub Actions** (не «Deploy from a branch»).
4. После успешного workflow сайт будет по адресу:

   **https://antoniozubakha.github.io/gift/**

Файл **`.nojekyll`** отключает Jekyll, чтобы статика отдавалась как есть.

## Локальный просмотр

```bash
python -m http.server 8080
```

Откройте http://localhost:8080/

## Настройки

В **`script.js`**: пароль подарка (`MAIN_PASSWORD`), `GIFT_EXTERNAL_URL`, `GIFT_DOWNLOAD_PATH`, пулы абсурдных реплик. Тексты загадок — в **`index.html`**.
